## 指尖方位代表含义

流连 速喜 赤口

大安 空亡 小吉

| 食指         | 中指         | 无名指      |
| ------------ | ------------ | ----------- |
| 指尖 -- 流连 | 指尖 -- 速喜 | 指尖 --赤口 |
|              |              |             |
| 指根 --大安  | 指根 --空亡  | 指根 --小吉 |

## 起卦方式

以心血来潮时的时间为准，拿农历月日时开始。

寅宫（左手食指根部）起月令。（以月开始顺时针方向算起，到本月时停止，在此地开始算日，再到结束时，以日的终点计算时辰）

如今日：七月初三十时多（亥时）左手食指开始，走七次到达大安，以大安开始走三次到达速喜，以速喜开始走 12 次到达流连处。获得结果为流连
