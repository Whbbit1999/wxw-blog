{% quot 近日喜欢的诗作 icon:hashtag %}

{% poetry 我咽下一枚铁做的月亮 author:许立志 footer:诗词节选 %}
我咽下一枚铁做的月亮
他们把它叫做螺丝
<br />
我咽下这工业的废水，失业的订单
那些低于机台的青春早早夭亡
<br />
我咽下奔波，咽下流离失所
咽下人行天桥，咽下长满水锈的生活
<br />
我再咽不下了
所有我曾经咽下的现在都从喉咙汹涌而出
在祖国的领土上铺成一首
耻辱的诗
{% endpoetry %}
